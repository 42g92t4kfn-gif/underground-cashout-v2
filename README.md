# Underground Cashout Feed Website

## Setup for Byethost

1. Create a MySQL database on Byethost
2. Import setup.sql
3. Edit config.php with your DB credentials
4. Upload all files
5. Create 'uploads' folder and set permission to 777
6. Default admin: username `admin` password `admin123`

Change password immediately after login.

Public feed: yoursite.com/index.php
Admin: yoursite.com/admin/dashboard.php
